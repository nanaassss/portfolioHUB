# 📚 Projetos e Atividades Acadêmicas

Este diretório contém os exercícios e estudos realizados durante as disciplinas do 1º semestre de ADS.

---

## 📋 Disciplinas e Atividades

### 💻 Introdução à Computação
- **Conteúdo**: Arquitetura de computadores, sistemas numéricos e fundamentos.
- **Atividades**: Exercícios de conversão de bases e estudos sobre hardware.

### ⚙️ Engenharia de Software
- **Conteúdo**: Introdução ao ciclo de vida de desenvolvimento e requisitos.
- **Atividades**: Documentação básica de ideias de software e fluxos de processos.

### 🧠 Lógica de Programação
- **Conteúdo**: Algoritmos, variáveis, condicionais e laços de repetição.
- **Atividades**: Pequenos scripts e algoritmos para resolução de problemas matemáticos e lógicos.

### 🗄️ Banco de Dados
- **Conteúdo**: Conceitos de bancos de dados relacionais e modelagem.
- **Atividades**: Diagramas Entidade-Relacionamento (DER) básicos e conceitos de tabelas.

---

## 🎯 Foco do Semestre

O foco principal deste semestre é construir a base teórica e lógica necessária para o desenvolvimento de sistemas complexos no futuro.
